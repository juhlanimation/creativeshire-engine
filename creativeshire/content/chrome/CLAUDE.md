# Chrome

**Site-wide persistent UI.**

- `regions/` — Header, Footer, Sidebar
- `overlays/` — Modal, Tooltip, CursorLabel

Before creating:
- Region or Overlay?
- Is it site-wide persistent UI, not page content?
- Is it generalized with configurable settings?
