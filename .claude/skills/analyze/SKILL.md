# Analyze

Analyze external references and create backlog items.

See [workflow.md](workflow.md) for detailed steps.

## Quick Reference

1. Fetch/view the reference
2. Identify components (widgets, sections, chrome, experiences)
3. Map to Creativeshire architecture
4. Create backlog items

## Output

Backlog items in `.claude/tasks/backlog.md` categorized by component type.
