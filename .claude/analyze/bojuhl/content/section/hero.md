# Hero Section

**Purpose:** Full-viewport intro with scroll-driven backgrounds
**Viewport:** 100vh

## Structure
```
HeroSection
├── BackgroundLayer (scroll-driven)
├── IntroText ("I'm Bo Juhl")
├── HeroTitle (EXECUTIVE PRODUCER / PRODUCER / EDITOR)
└── ScrollIndicator ("(SCROLL)")
```

## Widgets
- `HeroTitle`
- `IntroText`
- `ScrollIndicator`

## Behaviour
- `scroll-background`: Multiple illustrated backgrounds transition on scroll
